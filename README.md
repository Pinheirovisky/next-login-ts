# next-login-ts

Projeto com nextJs, com o intuito de funcionar como teste em um processo seletivo. As stacks foram criadas de acordo com as regras propostas, dentre elas:
  - NextJs;
  - TypeScript;
  - Jest;
  - Styled Components;
  - Redux Sagas
